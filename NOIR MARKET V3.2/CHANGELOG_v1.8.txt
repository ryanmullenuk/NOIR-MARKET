NOIR MARKET V1.8 CHANGELOG

Release type: Small bug-fix and usability update based on V1.7.

Changes made:
- Updated app version references to V1.8.
- Reworked the top news ticker so it uses smooth CSS animation instead of the old marquee behaviour.
- Added extra spacing between the yellow NEWS label and the moving ticker text so the words no longer overlap.
- Added a player name field at the top of the burger menu.
- Added Sounds ON/OFF toggle in the burger menu.
- Added Music ON/OFF toggle in the burger menu.
- Separated sound effects from background music controls.
- Expanded the Stats screen to include newer game systems:
  - Player name
  - Current rank
  - Best rank
  - Days held at rank
  - Reputation
  - Current city
  - Shipments exported
  - Shipments imported
  - Arrests
  - Jail days
  - Bribes paid
  - Informants paid
- Repositioned and spaced the Back to Menu button so it sits lower and does not overlap the stats content.
- Added Borrow Maximum Amount button to each loan shark screen.
- Borrow Maximum Amount fills the loan input with that lender’s current maximum permitted amount.
- Retained all V1.7 systems, including shipping, city vaults, dynamic economy, rumours, news events, proportional loan interest, music, particles and PWA icons.

Compatibility:
- V1.8 saves now use noir_market_v1_8.
- V1.7, V1.6, V1.5, V1.4, V1.3, V1.2 and older save fallback loading remains in place.

Checks completed:
- JavaScript syntax check passed.
- Manifest JSON validation passed.
- Service worker syntax check passed.
- ZIP integrity check passed.
