Noir Market V2.5 Performance Patch

Purpose
- Targeted splash/startup performance release.
- No gameplay, trading, market, storage, travel, shipping, loans, hustle, save or economy mechanics changed.

Changes
- Optimised the splash logo image from approximately 586 KB to approximately 30 KB.
- Prevented the MP3 music file from being precached by the service worker.
- Kept music loading until after user interaction rather than during initial install/cache.
- Locked splash particles to 100 active particles.
- Kept splash particles larger so the visual effect remains visible with fewer moving elements.
- Disabled the older startup routine that recreated the splash particles shortly after load.
- Confirmed game-page particles remain disabled.
- Added final V2.5 splash particle override using a document fragment to reduce DOM insertion cost.
- Updated title, manifest, cache name and save version to V2.5.

Compatibility
- V2.4 and earlier save migration retained.
- MP3 file remains in the package for music use, but is no longer part of the initial service worker cache.

Checks
- JavaScript syntax check passed.
- Manifest JSON validation passed.
- Service worker syntax check passed.
- ZIP integrity check passed.
